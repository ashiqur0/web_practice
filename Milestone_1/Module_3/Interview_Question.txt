1. What is Git? What are the advantages of using Git?
What is Git?
Git is a free and open-source distributed version control system. It allows developers to track changes to files, collaborate on projects, and manage different versions of code, making it a crucial tool for software development. 

What are the advantages of using Git?
Git offers numerous advantages, primarily in version control and collaboration. It allows developers to track changes, revert to previous versions, and work together on projects efficiently. Git's decentralized nature, branching capabilities, and focus on speed and security make it a powerful tool for modern software development.
